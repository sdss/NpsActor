Commands¶

The lvmnps replies to the follwing commands.

lvmnps¶

lvmnps [OPTIONS] COMMAND [ARGS]...

cycle¶

cycle power to an Outlet

lvmnps cycle [OPTIONS] [NAME] [PORTNUM]

Arguments

NAME¶

Optional argument

PORTNUM¶

Optional argument

help¶

Shows the help.

lvmnps help [OPTIONS] [PARSER_COMMAND]

Arguments

PARSER_COMMAND¶

Optional argument

off¶

Turn off the Outlet

lvmnps off [OPTIONS] [NAME] [PORTNUM]

Arguments

NAME¶

Optional argument

PORTNUM¶

Optional argument

on¶

Turn on the Outlet

lvmnps on [OPTIONS] [NAME] [PORTNUM]

Arguments

NAME¶

Optional argument

PORTNUM¶

Optional argument

onall¶

Turn on all Outlet

lvmnps onall [OPTIONS] [NAME]

Arguments

NAME¶

Optional argument

ping¶

Pings the actor.

lvmnps ping [OPTIONS]

status¶

print the status of the NPS.

lvmnps status [OPTIONS] COMMAND [ARGS]...

all¶

Returns the status of ALL outlets in the NPS.

lvmnps status all [OPTIONS]

what¶

Returns the status of the outlets.

lvmnps status what [OPTIONS] [NAME] [PORTNUM]

Arguments

NAME¶

Optional argument

PORTNUM¶

Optional argument

version¶

Reports the version.

lvmnps version [OPTIONS]
